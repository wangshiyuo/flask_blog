# blog


flask + sqlalchemy 搭建简易支持响应式页面个人博客

项目结构：

```
│  blog.py                        //项目入口文件
│  secure.py                      //安全配置信息
│  settings.py                    //项目配置信息
│  
└─app                             //后端APP
    |
    │
    │  
    ├─libs                        //自己编写的包
    │      
    ├─models                      //model文件
    │ 
    │      
    ├─static                      //前端css/js文件
    |                     
    |
    ├─templates                   //模板文件
    │      
    └─web                         //蓝图

```

具备功能：

        新建、编辑、删除文章，文章分类和查询、文章阅读排行、后台登陆
